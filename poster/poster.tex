\begin{tcolorbox}[posterbox={The Problem}]
  Students and residents searching for a rental apartment in Tartu must manually check
  \textbf{KV.ee}, \textbf{City24}, and \textbf{Rendin} separately — often multiple times
  per day. Listings disappear within hours and there is no way to track price changes
  or get notified when something affordable appears.
\end{tcolorbox}

\begin{tcolorbox}[posterbox={The Solution}]
  \textbf{Üüriturg} automatically scrapes all three platforms every \textbf{6 hours},
  deduplicates listings, and shows everything in one clean interface — with an
  \textbf{interactive map}, \textbf{price trend charts} per neighbourhood, and instant
  \textbf{email alerts} when a matching listing appears.
\end{tcolorbox}

\begin{tcolorbox}[posterbox={How It Works}]
  \begin{enumerate}[leftmargin=2cm, itemsep=5pt]
    \item \textbf{Scraper} fetches listings from KV.ee, City24 and Rendin
    \item \textbf{Deduplication} — same apartment never appears twice
    \item \textbf{RabbitMQ} delivers new listing events to alert service
    \item \textbf{Alert Service} matches listings against saved user criteria
    \item \textbf{Notification Service} sends email instantly via SMTP
  \end{enumerate}
\end{tcolorbox}

\begin{tcolorbox}[posterbox={Scraper Methods}]
  \begin{tabular}{>{\bfseries}p{3.5cm} p{10cm}}
    KV.ee    & HTML scraping via \texttt{wget} subprocess (bypasses Cloudflare) + Jsoup parsing \\[4pt]
    City24   & Public REST JSON API \texttt{api.city24.ee} \\[4pt]
    Rendin   & Firebase callable cloud function (POST) \\
  \end{tabular}
\end{tcolorbox}

\begin{tcolorbox}[posterbox={Key Features}]
  \begin{itemize}[leftmargin=2cm, itemsep=6pt]
    \item \textbf{Interactive Map} — live listings with cluster markers across Tartu
    \item \textbf{Price Trend Charts} — average rent over time by neighbourhood
    \item \textbf{Email Alerts} — get notified instantly when a match appears
    \item \textbf{Auto-Scraping} — KV.ee, City24, Rendin every 6 hours
    \item \textbf{Admin Dashboard} — trigger scrapes, monitor system health
    \item \textbf{Deduplication} — same listing never shown twice
    \item \textbf{10 Neighbourhoods} — Kesklinn, Karlova, Annelinn, Supilinn...
  \end{itemize}
\end{tcolorbox}

  \renewcommand{\arraystretch}{1.4}

    Backend     & Java 21 + Spring Boot 3.4.5 (8 microservices) \\

    Database    & PostgreSQL 16 (one schema per service) \\
    Messaging   & RabbitMQ 3 via Spring AMQP \\
